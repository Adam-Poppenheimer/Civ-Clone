IBorderExpansionLogic
Defines the process by which cities acquire new cells. This class specifies the amount of culture or gold required to acquire a new cell, what cells are available for acquisition, and what cell a city should actively pursue.
ICellPossessionCanon
Defines and controls the ownership of cells by cities, which determines which cells are within which cities' borders.
