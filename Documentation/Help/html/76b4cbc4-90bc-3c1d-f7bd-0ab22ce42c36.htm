The base interface for all building factories.
IBuildingPossessionCanon
The canon for building possession by cities. This class determines what buildings belong to a city and whether a building can be placed in a city. It also provides methods for adding and removing buildings from city.
IBuildingProductionValidityLogic
Determines what templates are valid for construction in various cities.

The base interface for all building templates, which define what buildings are available for construction in various cities.
