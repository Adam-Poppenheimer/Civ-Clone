Assets.Simulation.Civilizations Namespace

Contains all the code that defines, creates, and executes civilizations, which are the largest actors in the game and represent all the things that a particular player has control over.

Interfaces
  InterfaceDescription
ICivilization
The base interface for civilizations, which represent the largest agent in the game. Every player controls a single civilization, which in turn controls a number of cities and units.
ICivilizationConfig
Defines a variety of fields that configure the behavior of Civilizations and other similar classes.
ICivilizationFactory
Defines a factory capable of creating civilizations of a particular name.
