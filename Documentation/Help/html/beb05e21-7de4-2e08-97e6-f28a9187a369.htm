Assets.Simulation.Core Namespace

Contains classes that lie at the core of gameplay execution. Mostly involves player input and turn execution.
Classes
  ClassDescription
CityBuilder
A temporary utility used to build cities to empty maps. Should not remain in the build much longer.
CoreInstaller
The installer that managed dependency injection for all classes and signals associated with the Core namespace
EndTurnRequestedSignal
A signal that fires whenever the current turn should be ended and a new one began.
GameCore
Controls the chronological progression and other core elements of the game. Currently, that means handling turn incrementation and keeping a reference to the player civilization.
PlayerInputInstaller
The installer that initializes and binds all player input signals, which mostly includes events that can't be caught or heard by other classes.
TurnBeganSignal
A signal that should fire whenever a new turn is begun.
TurnEndedSignal
A signal that should fire whenever the old turn has ended.
TurnExecuter
The standard implementation of ITurnExecuter.
Interfaces
  InterfaceDescription
ITurnExecuter
Performs the tasks that must occur at the beginning and end of turns for various classes.
