Remarks
Unlike other factory classes, this factory checks both technical and mechanical restrictions to city placement. It calls into IBuildingProductionValidityLogic.IsTemplateValidForCity and IBuildingPossessionCanon.GetBuildingsInCity to make sure that the argued template is valid. This inconsistency should probably be resolved, though it's not clear whether BuildingFactory is a model or an aberration.
